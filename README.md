# Personal Portfolio Website

A clean, modern, and responsive personal portfolio website designed to showcase my software development projects, skills, and professional journey. 

🌐 **Live Demo Link:**   [Portfolio - Kinzi Charles](https://kinzicharles.vercel.app)

## 🚀 Features

*   **Responsive Design:** Fully optimized for desktop, tablet, and mobile screens.
*   **Project Showcase:** Interactive display of featured development projects.
*   **Contact System:** Integrated backend API support to handle contact form inquiries.
*   **Modern UI/UX:** Smooth scrolling, clean typography, and interactive hover states.

## 🛠️ Tech Stack

*   **Frontend:** HTML5, CSS3, JavaScript (ES6+)
*   **Deployment:** [Vercel](https://vercel.com)
*   **Environment:** Node.js (Package management)

## 🌐 Deployment

This project is configured for automated deployment via **Vercel**. Every push to the `main` branch automatically triggers a new live build.
